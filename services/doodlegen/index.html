One-Time Setup

Takes ~5 minutes. After this, sign in once and tokens refresh automatically forever.

  1. 1
    Go to console.cloud.google.com → Create a new project (or pick an existing one)
  2. 2
    APIs & Services → OAuth consent screen
    User Type: External → Fill in app name (e.g. "DoodleGen") → Save.
    Under Scopes, add: cloud-platform
    Under Test users, add your own Google email.
  3. 3
    APIs & Services → Credentials → Create Credentials → OAuth client ID
    Application type: Web application
    Authorized redirect URI: http://localhost:3000/auth/callback
  4. 4
    Copy your Client ID and Client Secret.
    Create oauth.json in the same folder as server.js: 
    {
  "client_id": "YOUR_CLIENT_ID.apps.googleusercontent.com",
  "client_secret": "YOUR_CLIENT_SECRET"
}
  5. 5
    Restart the server (node server.js), then come back and click Sign in with Google above. Done — you'll never need to paste a token again.
Already have a bearer token? You can still paste it manually via the Tokens button in the nav. Manual tokens work as a fallback if OAuth is not configured.
